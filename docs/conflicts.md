# Работа с конфликтами

Если файл исправленный пользователем без сохранения изменён вне редактора, OPFS Studio показывает баннер конфликта.

## Варианты действий

- **Откатить изменения** — сбросить локальные правки и загрузить версию с диска.
- **Заменить содержимым диска** — перезаписать файл в OPFS локальными изменениями.
- **Оставить мои правки** — игнорировать внешние изменения.
- **Смерджить** — открыть визуальное сравнение и объединить правки.

![Конфликт изменений](./screenshots/conflict.png)

## Сравнение изменений

Приложение открывает diff-просмотрщик, подсвечивая удалённые/добавленные строки.
