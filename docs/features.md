# Функциональность

## Работа с файлами

- Создание новых файлов с указанием пути.
- Редактирование в текстовом редакторе.
- Сохранение изменений в OPFS.

## Работа с папками

- Создание папок, включая вложенные (`a/b/c`).
- Удаление и переименование.

## Поддерживаемые форматы

- Текстовые (txt, json, xml, md, js, ts и другие).
- Изображения (jpg, png, gif).
- PDF.
- Бинарные файлы (загружаются, но не редактируются).

## Дополнительно

- История редактирования (откат несохранённых правок).
- Статус-бар с индикатором изменений.
