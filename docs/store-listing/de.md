# Chrome Web Store — Deutsch

**Title** (≤45)
OPFS Studio — Datei-IDE für den Browser

**Summary** (≤132)
Durchsuchen, bearbeiten und verwalten Sie das Origin Private File System (OPFS) — eine Datei-IDE mit Monaco-Editor, Live-Sync und Vorschauen.

**Category:** Developer Tools

---

**Detailed description**

Wollten Sie schon einmal in das Origin Private File System schauen, in das Ihre
Web-App schreibt — und mussten feststellen, dass die DevTools es einfach nicht
anzeigen? OPFS Studio ist der fehlende Dateimanager und Code-Editor für OPFS,
direkt im Browser.

Öffnen Sie das Seitenpanel (oder den DevTools-Tab) und erhalten Sie sofort eine
Live-Baumansicht aller Dateien und Ordner, die Ihre Seite im OPFS speichert.

★ FUNKTIONEN

• Vollständiger Dateibaum — verschachtelte Ordner, Dateien, Bilder, PDFs
• Monaco-Editor — dieselbe Engine wie VS Code: Syntaxhervorhebung, Formatierung
• Live-Überwachung — externe Änderungen sofort sichtbar, mit intelligenter
  Konflikterkennung
• Konfliktlösung — visueller Side-by-Side-Diff mit Merge-Optionen
• Vorschauen — Tabellenansicht für CSV/TSV, Markdown-Rendering, SVG-Split-View,
  Vorschau für Bilder / Video / Audio / PDF
• Erstellen, umbenennen, verschieben, löschen — Dateien und ganze Ordnerbäume
• Hochladen & Herunterladen
• Dunkles & helles Design
• Drei Oberflächen — Seitenpanel, Popup und ein DevTools-Panel
• 7 UI-Sprachen

★ DATENSCHUTZ

Ihre Dateien verlassen niemals den Browser. OPFS Studio lädt, teilt oder
speichert Ihre Dateiinhalte nirgendwo.

Installieren Sie OPFS Studio und sehen — und steuern — Sie endlich, was Ihr
Browser speichert.
