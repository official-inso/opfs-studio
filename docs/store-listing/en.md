**Title** (≤45 chars)
OPFS Studio — Browser File System IDE

**Summary / short description** (≤132 chars)
Browse, edit and manage the browser's Origin Private File System (OPFS) — a full file IDE with Monaco editor, live sync and previews.

**Category:** Developer Tools

---

**Detailed description**

Ever needed to peek inside the Origin Private File System your web app writes
to — and found that DevTools simply can't show it? OPFS Studio is the missing
file manager and code editor for OPFS, right inside your browser.

Open the side panel (or the DevTools tab) and instantly get a live, tree-style
view of every file and folder your site keeps in OPFS. Inspect, edit and
organize them like a real IDE — no rebuilds, no guesswork.

★ WHAT YOU GET

• Full file tree — browse nested folders, files, images, PDFs and more
• Monaco editor — the same engine as VS Code: syntax highlighting, multi-file
  editing, formatting
• Live watch — see external changes the moment they happen, with smart
  conflict detection (no false positives on identical content)
• Conflict resolution — a visual side-by-side diff with merge, keep-mine or
  take-disk options when a file changes outside the editor
• Rich previews — built-in table view for CSV/TSV, rendered Markdown preview,
  split-view SVG (code + render), and image / video / audio / PDF preview
• Create, rename, move, delete — files and whole nested folder trees
• Upload & download — drop files and folders into OPFS, export any file out
• Dark & light themes — instant, no reload
• Three surfaces — side panel, action popup, and a dedicated DevTools panel
• 7 UI languages — English, Русский, Français, Deutsch, Español, 中文, 日本語

★ WHO IT'S FOR

Web developers and frontend engineers building offline-first apps, working with
WASM/SQLite-in-the-browser, caching layers, or anything that uses the Origin
Private File System. Students and researchers exploring modern Web Storage APIs.

★ PRIVACY FIRST

Your files never leave the browser. OPFS Studio doesn't upload, share or store
your file contents anywhere. See our privacy policy for details.

Install OPFS Studio and finally see — and control — what your browser is
storing.
