# Chrome Web Store — Español

**Title** (≤45)
OPFS Studio — IDE de archivos del navegador

**Summary** (≤132)
Explora, edita y administra el Origin Private File System (OPFS): un IDE de archivos con editor Monaco, sincronización en vivo y vistas previas.

**Category:** Developer Tools

---

**Detailed description**

¿Alguna vez quisiste mirar dentro del Origin Private File System donde escribe
tu aplicación web y descubriste que las DevTools simplemente no lo muestran?
OPFS Studio es el administrador de archivos y editor de código que falta para
OPFS, directamente en el navegador.

Abre el panel lateral (o la pestaña de DevTools) y obtén al instante una vista
en árbol y en vivo de todos los archivos y carpetas que tu sitio guarda en OPFS.

★ CARACTERÍSTICAS

• Árbol de archivos completo — carpetas anidadas, archivos, imágenes, PDF
• Editor Monaco — el mismo motor que VS Code: resaltado de sintaxis, formato
• Observación en vivo — cambios externos visibles al instante, con detección
  inteligente de conflictos
• Resolución de conflictos — diff visual lado a lado con opciones de fusión
• Vistas previas — vista de tabla para CSV/TSV, render de Markdown, vista
  dividida de SVG, vista previa de imagen / vídeo / audio / PDF
• Crear, renombrar, mover, eliminar — archivos y árboles de carpetas completos
• Subir y descargar
• Temas oscuro y claro
• Tres superficies — panel lateral, popup y panel de DevTools
• 7 idiomas de interfaz

★ PRIVACIDAD

Tus archivos nunca salen del navegador. OPFS Studio no sube, comparte ni
almacena el contenido de tus archivos en ningún lugar.

Instala OPFS Studio y por fin ve — y controla — lo que tu navegador almacena.
