# Chrome Web Store — Français

**Title** (≤45)
OPFS Studio — IDE de fichiers navigateur

**Summary** (≤132)
Explorez, modifiez et gérez l'Origin Private File System (OPFS) : un IDE de fichiers avec éditeur Monaco, synchro en direct et aperçus.

**Category:** Developer Tools

---

**Detailed description**

Vous avez déjà voulu regarder dans l'Origin Private File System où votre
application web écrit — pour découvrir que les DevTools ne l'affichent tout
simplement pas ? OPFS Studio est le gestionnaire de fichiers et l'éditeur de
code manquant pour OPFS, directement dans le navigateur.

Ouvrez le panneau latéral (ou l'onglet DevTools) et obtenez instantanément une
arborescence en direct de tous les fichiers et dossiers que votre site conserve
dans OPFS.

★ FONCTIONNALITÉS

• Arborescence complète — dossiers imbriqués, fichiers, images, PDF
• Éditeur Monaco — le même moteur que VS Code : coloration syntaxique, formatage
• Surveillance en direct — changements externes visibles immédiatement, avec
  détection intelligente des conflits
• Résolution de conflits — diff visuel côte à côte avec options de fusion
• Aperçus — vue tableau pour CSV/TSV, rendu Markdown, vue partagée SVG,
  aperçu images / vidéo / audio / PDF
• Créer, renommer, déplacer, supprimer — fichiers et arborescences entières
• Importer & exporter
• Thèmes sombre & clair
• Trois surfaces — panneau latéral, popup et panneau DevTools
• 7 langues d'interface

★ CONFIDENTIALITÉ

Vos fichiers ne quittent jamais le navigateur. OPFS Studio ne téléverse, ne
partage ni ne stocke le contenu de vos fichiers nulle part.

Installez OPFS Studio et voyez — et contrôlez — enfin ce que votre navigateur
stocke.
