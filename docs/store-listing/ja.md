# Chrome Web Store — 日本語

**Title** (≤45)
OPFS Studio — ブラウザのファイルIDE

**Summary** (≤132)
ブラウザの Origin Private File System (OPFS) を閲覧・編集・管理。Monacoエディタ、ライブ同期、プレビュー搭載のファイルIDE。

**Category:** Developer Tools

---

**Detailed description**

ウェブアプリが書き込む Origin Private File System の中を見たいのに、DevToolsでは
表示できなくて困ったことはありませんか？ OPFS Studio は、ブラウザ内で動作する
OPFS のためのファイルマネージャ兼コードエディタです。

サイドパネル（または DevTools タブ）を開くだけで、サイトが OPFS に保存している
すべてのファイルとフォルダをライブのツリー表示で確認できます。

★ 主な機能

• 完全なファイルツリー — ネストしたフォルダ、ファイル、画像、PDF
• Monaco エディタ — VS Code と同じエンジン：シンタックスハイライト、整形
• ライブ監視 — 外部の変更を即座に検知（同一内容での誤検知なし）
• 競合の解決 — 並列差分表示とマージ／自分の変更を保持／ディスクを採用
• 豊富なプレビュー — CSV/TSV のテーブル表示、Markdown レンダリング、
  SVG の分割表示、画像 / 動画 / 音声 / PDF プレビュー
• 作成・名前変更・移動・削除 — ファイルとフォルダツリー全体
• アップロードとダウンロード
• ダーク／ライトテーマ
• 3つの画面 — サイドパネル、ポップアップ、専用 DevTools パネル
• 7言語のUI

★ プライバシー

ファイルがブラウザの外に出ることはありません。OPFS Studio はファイルの内容を
どこにもアップロード・共有・保存しません。

OPFS Studio をインストールして、ブラウザが保存している内容をついに確認・管理
しましょう。
