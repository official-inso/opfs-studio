# Chrome Web Store — Русский

**Title** (≤45)
OPFS Studio — IDE для файлов браузера

**Summary** (≤132)
Просмотр, редактирование и управление Origin Private File System (OPFS): полноценная IDE с редактором Monaco, синхронизацией и превью.

**Category:** Developer Tools

---

**Detailed description**

Нужно было заглянуть в Origin Private File System, куда пишет ваше веб-приложение,
а DevTools его попросту не показывает? OPFS Studio — это недостающий файловый
менеджер и редактор кода для OPFS прямо в браузере.

Откройте боковую панель (или вкладку в DevTools) и сразу получите живое
дерево всех файлов и папок, которые ваш сайт хранит в OPFS. Просматривайте,
редактируйте и организуйте их как в настоящей IDE — без пересборок и догадок.

★ ВОЗМОЖНОСТИ

• Полное файловое дерево — вложенные папки, файлы, изображения, PDF и не только
• Редактор Monaco — тот же движок, что в VS Code: подсветка синтаксиса,
  многофайловое редактирование, форматирование
• Live-наблюдение — внешние изменения видны мгновенно, с умным определением
  конфликтов (без ложных срабатываний на одинаковом содержимом)
• Разрешение конфликтов — визуальное сравнение с вариантами «слить»,
  «оставить мои правки» или «взять с диска»
• Богатые превью — табличный просмотр CSV/TSV, рендер Markdown, split-view
  для SVG (код + рендер), превью изображений / видео / аудио / PDF
• Создание, переименование, перемещение, удаление — файлов и целых деревьев
• Загрузка и скачивание — перетаскивайте файлы и папки в OPFS, выгружайте обратно
• Тёмная и светлая темы — мгновенно, без перезагрузки
• Три поверхности — боковая панель, popup и отдельная панель DevTools
• 7 языков интерфейса — English, Русский, Français, Deutsch, Español, 中文, 日本語

★ ДЛЯ КОГО

Веб-разработчики и фронтенд-инженеры, делающие offline-first приложения,
работающие с WASM/SQLite в браузере, слоями кеширования — всем, кто использует
Origin Private File System. А также студенты и исследователи современных Web
Storage API.

★ ПРИВАТНОСТЬ

Ваши файлы никогда не покидают браузер. OPFS Studio не загружает, не передаёт
и нигде не хранит содержимое ваших файлов. Подробности — в политике
конфиденциальности.

Установите OPFS Studio и наконец-то увидьте — и контролируйте — что хранит ваш
браузер.
