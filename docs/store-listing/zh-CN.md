# Chrome Web Store — 中文（简体）

**Title** (≤45)
OPFS Studio — 浏览器文件 IDE

**Summary** (≤132)
浏览、编辑和管理浏览器的 Origin Private File System (OPFS)：内置 Monaco 编辑器、实时同步与预览的文件 IDE。

**Category:** Developer Tools

---

**Detailed description**

是否曾想查看你的 Web 应用写入的 Origin Private File System，却发现 DevTools 根本
无法显示？OPFS Studio 正是浏览器中缺失的 OPFS 文件管理器和代码编辑器。

打开侧边栏（或 DevTools 标签页），即可即时获得网站存储在 OPFS 中的所有文件和
文件夹的实时树状视图。

★ 功能

• 完整文件树 — 嵌套文件夹、文件、图片、PDF 等
• Monaco 编辑器 — 与 VS Code 相同的引擎：语法高亮、格式化
• 实时监视 — 立即显示外部更改，并智能检测冲突（相同内容不误报）
• 冲突解决 — 可视化并排差异，支持合并 / 保留我的修改 / 采用磁盘版本
• 丰富预览 — CSV/TSV 表格视图、Markdown 渲染、SVG 分屏（代码 + 渲染）、
  图片 / 视频 / 音频 / PDF 预览
• 创建、重命名、移动、删除 — 文件和整个文件夹树
• 上传与下载
• 深色与浅色主题
• 三种界面 — 侧边栏、弹出窗口和专用 DevTools 面板
• 7 种界面语言

★ 隐私

你的文件永远不会离开浏览器。OPFS Studio 不会上传、共享或在任何地方存储你的
文件内容。

安装 OPFS Studio，终于看见并掌控浏览器存储的内容。
